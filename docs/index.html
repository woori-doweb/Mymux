// features

Everything in one window

한 창 안에서 모든 것을 — 셸부터 브라우저까지.

⌨
Terminal
// terminal

Tabs and horizontal/vertical split panes. Drag panes to rearrange. Each pane shows its working directory. Choose your default shell: Git Bash, PowerShell, or CMD.

탭 + 가로/세로 분할 패인, 드래그 재배치, 패인별 작업 디렉터리 표시. 기본 셸 Git Bash / PowerShell / CMD 선택.

split-pane git-bash powershell cmd
⚡
Multiplexer power tools
// power_tools

Scrollback search (Ctrl+Shift+F), pane zoom (Ctrl+Shift+Z), and input broadcast (Ctrl+Shift+B, tmux synchronize-panes) across a tab. Ctrl+Click opens links in the in-app browser. A hidden pane that produces output or finishes a task dots its session and tab, and flashes the taskbar when the window is in the background.

스크롤백 검색(Ctrl+Shift+F), 패인 최대화(Ctrl+Shift+Z), 탭 단위 입력 브로드캐스트(Ctrl+Shift+B). Ctrl+클릭으로 링크를 내장 브라우저에서 열고, 숨은 패인의 출력/작업 완료는 세션·탭 배지와 작업표시줄 깜빡임으로 알림.

search zoom broadcast activity-badge clickable-links
🔒
SSH / SFTP
// ssh_sftp

Connect to remote servers via SSH using password or key authentication. Browse and manage remote files directly through the built-in SFTP explorer. Save connections for one-click reconnect, and attach straight into a tmux session on connect.

SSH(비밀번호·키) 접속 + 내장 SFTP로 원격 파일 탐색. 접속 정보 저장으로 재접속, 연결 시 tmux 세션 바로 연결.

ssh sftp password-auth key-auth tmux saved-hosts
📁
File Explorer
// file_explorer

Drive list, favorites, manual path input, and folder tree. Back/forward history, parent folder navigation, and Chrome-style mouse side buttons. Type cd to open a terminal in the current folder.

드라이브·즐겨찾기·경로 입력·폴더 목록, 히스토리 탐색, 마우스 옆 버튼, cd로 그 폴더에서 터미널 열기.

favorites history cd
📄
File Viewer
// file_viewer

Renders Markdown, plain text, and code with syntax highlighting. Sandboxed HTML preview. Multiple open file tabs. Right-click selected text to copy or send directly to a terminal session.

마크다운/텍스트/코드 렌더링, HTML 샌드박스 미리보기, 여러 파일 탭. 선택 영역 우클릭 → 복사/세션으로 보내기.

markdown syntax-highlight html-sandbox
🌐
In-app Browser
// in_app_browser

Native mode — address bar, back/forward, and refresh, just like a real browser.
AI Control mode — exposes a CDP endpoint so Playwright or MCP can drive it programmatically.

Native 모드(주소창·앞/뒤·새로고침) + AI 제어 모드(CDP 엔드포인트로 Playwright/MCP 조종).

cdp playwright mcp native
✨
Quality of Life
// convenience

Saved commands, session save & restore on exit, light/dark theme with a custom accent color picker. Smart clipboard (Ctrl+C/V, OSC 52, IME-safe, drag-to-copy, image paste) and adjustable font size & letter spacing. Auto-update via GitHub Releases.

저장 명령, 종료 시 세션 저장·복원, 라이트/다크 테마 + 강조색, 스마트 클립보드(Ctrl+C/V·OSC 52·한글 IME 안전·드래그 복사·이미지 붙여넣기), 글자 크기·자간 조절, 자동 업데이트(GitHub Releases).

session-restore dark-mode auto-update accent-color clipboard ime-safe
🔔
Task-done Notification
// task_done_flash

When a long command finishes (OSC 133) or output goes quiet — even bursty output with gaps, like plugin updates — the pane border and session row pulse for 10 seconds — scarlet by default, hue-shifted per session so parallel jobs are tellable apart. Hovering the pane dismisses it; short commands and typing never trigger it.

오래 걸린 명령이 끝나거나 출력이 멈추면(플러그인 업데이트처럼 띄엄띄엄 출력하는 작업 포함) 패인 외곽선·세션 행이 10초간 펄스 — 기본 다홍색, 세션마다 색조가 달라 병렬 작업도 한눈에 구분. 마우스를 올리면 바로 해제.

osc-133 notification per-session border-flash hover-dismiss
